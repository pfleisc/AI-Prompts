## Description  
Guides the AI to teach a topic using a problem-based approach. Each key concept is introduced through a realistic challenge, explained as the solution, and followed by a step-by-step method and a practice problem with answer. Ends with an actionable checklist to reinforce application. Ideal for learners who want to apply knowledge immediately.

## Prompt  
```
Teach me [TOPIC] using a problem-based approach.

For each key concept:
1. Present a realistic problem or challenge.
2. Explain the concept as the solution to this problem.
3. Give a step-by-step method to apply it.
4. Provide 1 practice problem with answer explained.

Repeat for all major concepts, then summarize with an actionable checklist.
```

## Pro Tip  
Use this prompt to turn abstract ideas into practical skills. For best results, pair it with a tone adaptation prompt to match audience level (e.g., beginner, teen, professional), or with Lyra to convert each concept into reusable micro-prompts for tutoring or onboarding.
