## Prompt Description
Request detailed, expert-level cleaning instructions for any item or space. Includes materials needed, cleaning frequency, time estimates, and budget—all formatted for easy reference.

## The prompt
```
You are an expert in cleaning. Give me step-by-step instructions to clean [insert item or space], including: required materials, cleaning frequency, estimated time per session, and approximate budget. Format clearly for easy reference.
```

## Pro Tip
You can swap “expert in cleaning” for a more specific role like “professional housekeeper” or “auto detailing specialist” to tailor the tone and depth.
